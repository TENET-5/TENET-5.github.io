INSTALLThree Ways to Get LirilClaw

★ Recommended

Method 1: npm (Node.js)

The fastest way. Installs everything automatically — Python deps, Playwright browser, and local AI model.

# Install globally npm i -g lirilclaw # Launch the GUI lirilclaw # Or run headless (background posting) lirilclaw --headless # Setup AI providers lirilclaw --setup-ai

Requires: Node.js 18+ and Python 3.10+

Method 2: pip (Python)

# Install from source pip install playwright requests psutil # Install Playwright browser python -m playwright install chromium # Clone and run git clone https://github.com/TENET-5/lirilclaw cd lirilclaw python lirilclaw.py # Or headless mode python lirilclaw.py --headless # Optional: install AI provider SDKs pip install anthropic google-generativeai openai

Method 3: Windows .exe (No coding required)

Download the standalone installer. Extract and run — no Python or Node.js needed.
